# Hand Gesture Tic-Tac-Toe

A Computer Vision based Tic-Tac-Toe game built using Python, OpenCV, and CVZone.

## Features
- Hand gesture control
- AI opponent using Minimax Algorithm
- Real-time finger tracking
- Play Again button
- Unbeatable AI

## Technologies
- Python
- OpenCV
- CVZone
- MediaPipe

## Run

pip install -r requirements.txt

python tic_tac_toe.py
